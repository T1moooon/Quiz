# Викторина в Telegram и VK

В проекте реализованы два бота для Telegram и ВКонтакте. Они читают подборку вопросов из папки `quiz-questions`, сохраняют активные задания в Redis и разделяют общую логику через модуль `quiz_utils.py`. Пользователь может запросить новый вопрос, ответить на него или сдаться — оба бота ведут себя одинаково.

## Примеры ботов

[VK бот](https://vk.com/club232680346)

[TG бот](https://t.me/Timontester_bot)

## Установка и запуск

1. Убедитесь, что установлен **Python 3.12**.
2. Клонируйте репозиторий.
3. Создайте виртуальное окружение:
   ```bash
   python3 -m venv venv
   ```
4. Активируйте окружение:
   - Windows  
     ```bash
     venv\Scripts\activate
     ```
   - Linux/macOS  
     ```bash
     source venv/bin/activate
     ```
5. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
6. Убедитесь, что локальный или удалённый Redis доступен.

## Переменные окружения

В корне проекта создайте файл `.env` и пропишите:

```ini
TG_BOT_TOKEN=токен_телеграм_бота
VK_BOT_TOKEN=токен_vk_бота
REDIS_URL=redis://localhost:6379/0
```

- **TG_BOT_TOKEN** — токен от [BotFather](https://t.me/BotFather).
- **VK_BOT_TOKEN** — ключ доступа ВКонтакте. Чтобы его получить:  
  - Создайте сообщество (если ещё не создано).  
  - Зайдите в раздел **Управление → Дополнительно → Работа с API**.  
  - Нажмите **Создать ключ**. 
- **REDIS_URL** — строка подключения к Redis; можно оставить значение по умолчанию.

## Запуск ботов

Каждый бот запускается в отдельном терминале:

- Telegram:
  ```bash
  python tg_bot.py
  ```
- VK:
  ```bash
  python vk_bot.py
  ```
